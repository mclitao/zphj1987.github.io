centos做时间同步服务器

修改同步服务器的配置文件/etc/ntp.conf ,删除所有的内容,添加

restrict default nomodify
server  127.127.1.0     # local clock
fudge   127.127.1.0 stratum 8

重启ntpd

/etc/init.d/ntpd restart

等待ntp服务器自身同步完成

watch ntpq -p

观察这个reach需要超过17
否则其他机器ntpdate同步的时候会报错
