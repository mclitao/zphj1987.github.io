linux利用screen进行shell下的屏幕协作

我们都知道linux是支持多终端并行处理的
但是某些时候我们可能有比较特殊的需求需要两个人同时处理一个终端,screen 正好能满足这个要求

首先需要安装screen软件:

debian和ubuntu下面都是通过:

apt-get install screen

进行安装的.

假如现在有a和b需要进行屏幕协作,那么操作步骤如下:
a在终端运行:

screen -S   zp

b然后在终端运行: 

screen -x  zp

这个时候两边的用户的操作是实时同步的。
