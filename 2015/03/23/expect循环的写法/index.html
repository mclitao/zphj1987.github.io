expect循环的写法

脚本:

#!/usr/bin/expect
spawn sensors-detect
for {set i 0} {$i<=10} {incr i} {
expect ":"
send "\n"
}
interact

解释:

spawn是启动命令
for为循环的写法
interact为退出
这个脚本目的是在一次运行过程中不断模拟用户的enter操作
